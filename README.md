# Algoritmos Bubble e Selection Sort em C

Programa em C para análise de desempenho dos algoritmos Bubble Sort e Selection Sort,onde ocorre as
comparações entre o tempo de execução, número de comparações e movimentações entre eles.

## Algoritmos analisados
- Bubble Sort
- Selection Sort

## Complexidade
Ambos possuem complexidade O(n²)

## Metodologia utilizada
- Vetores de tamanho: 100, 1000 e 10000
- Dados em ordem decrescente (pior caso)
- Medição de tempo com clock()

## Conclusão
O Selection Sort apresentou melhor desempenho por realizar menos movimentações que o Bubble Sort.
